# 易宿酒店预订平台 - 前端开发计划

## 开发策略

采用**渐进式开发**策略：每完成一个功能模块，立即进行测试验证，确保功能正常后再进入下一个模块。

## 阶段二：用户端（功能复杂，需要更多时间）

### Phase 2.1：项目初始化

**任务清单：**
- [x] 初始化 Taro 项目
- [x] 配置多端编译（微信小程序、H5）
- [x] 安装依赖
- [ ] 配置 API 请求封装
- [ ] 创建基础路由

**开发流程：**

1. **开发模式**
   ```bash
   cd client-weapp-h5
   npm run dev:h5        # H5 开发模式，自动启动服务器（默认 http://localhost:10086）
   npm run dev:weapp     # 微信小程序开发模式
   ```
   - 支持热更新，修改代码自动刷新
   - 自动启动开发服务器

2. **生产构建**
   ```bash
   npm run build:h5      # 构建 H5 版本到 dist/ 目录
   npm run build:weapp   # 构建微信小程序版本到 dist/ 目录
   ```

3. **预览构建产物**
   ```bash
   # 先构建
   npm run build:h5

   # 再启动静态服务器
   cd dist
   python -m http.server 8000
   # 访问 http://localhost:8000
   ```

**测试验证：**
- [x] 项目可以正常编译
- [ ] 可以在微信开发者工具中预览
- [x] H5 版本可以在浏览器中打开

## 技术栈确认

### 管理后台
- **框架**: React 18 + TypeScript
- **UI 库**: Ant Design 5.x
- **路由**: React Router 6
- **状态管理**: React Context / Zustand（可选）
- **HTTP 请求**: axios
- **构建工具**: Vite

### 用户端
- **框架**: Taro 3.x + React + TypeScript
- **UI 库**: Taro UI / NutUI（可选）
- **状态管理**: Zustand / Redux Toolkit
- **HTTP 请求**: Taro.request 封装
- **多端支持**: 微信小程序 + H5
